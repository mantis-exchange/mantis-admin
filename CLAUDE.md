# mantis-admin

Mantis Exchange admin dashboard.

## Pages

- **Dashboard** — System status overview, service health checks
- **Users** — User management, KYC review, account freeze
- **Withdrawals** — Withdrawal approval queue
- **Trading Pairs** — Add/remove/halt trading pairs

## Tech Stack

- React + TypeScript + Vite
- react-router-dom
- axios

## Development

```bash
npm install
npm run dev    # http://localhost:5174
npm run build
```
